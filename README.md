# Before you run the test script:
Before you can run the tests you have to provide your own Trello API key and token.

## Add your api key and token in the config file
- Add your API key in the ./test/configs/environment.config.js file
- Add your API token in the ./test/configs/environment.configs.js file

## Create a dotenv file
- Create a .env file in the root directory of this project
- Add your API key as: API_KEY="your_api_key"
- Add your API token as: API_TOKEN="your_api_token"
